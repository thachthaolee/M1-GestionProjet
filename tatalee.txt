Github.com:
- créer compte
- créer nouveau Repository
   ->.gitignore template = Python
   -> privé
-> Clone Repo: git clone "lien du projet"
-> Commit: créer le fichier dans projet locale puis
	   git status
         git add "nom du fichier"
	   git commit -m "description du commit"
-> Push : git push
->log en ligne : voir s'il est mise à jour en ligne
